## Commands
### namedlocation
| command        | desc                    | 
| -------------- | ----------------------- | 
| nl add <label> | add location as <label> | 
| nl rm <label>  | remove location <label> | 
| nl tp <label>  | tp to location <label>  | 
| nl ls          | list labeled locations  |
